FAQ

The honest answers.

Does it send my code anywhere?
No — by default everything runs locally and offline: no network calls, no telemetry, no account. The one exception is a hosted LLM you explicitly opt into at setup (local Ollama stays fully offline).
Does it cost anything?
No — $0 by default. Local embeddings, no API keys required. abs setup offers a guided, skippable LLM step that sharpens recall — local Ollama is $0 and offline, or any hosted OpenAI-compatible endpoint; your key is never stored. Skip it and recall still works.
Which agents does it work with?
Five harnesses — Claude Code, Codex CLI, Gemini CLI, GitHub Copilot CLI and OpenCode — each via MCP, with hands-free session capture and context injection through that harness's native lifecycle. The same local memory is shared across all of them.
Is it open source?
Fully — on GitHub. Fork it, read every line.
